Game instructions:

Color Maze is an exciting puzzle game where you control a character through a maze. The goal is to color all the tiles by guiding the character over them. But be careful! There are holes in the ground that you must avoid, otherwise you'll have to start over. The character can only move in four directions: up, down, left, and right. Each time you complete a level, a new maze is generated for you to explore.

Lvl: 1/50
TIME: 10s
